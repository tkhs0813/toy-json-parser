# Toy JSON Parser in TypeScript

TypeScript を学習するための JSON パーサー実装プロジェクトです。TDD（テスト駆動開発）の手法を用いて、JSON パーサーを一から実装していきます。

## プロジェクト構造

```
toy-json-parser-ts/
  ├── src/
  │   ├── index.ts          # メインのエクスポートポイント
  │   ├── parser.ts         # JSONパーサーの実装
  │   └── parser.test.ts    # JSONパーサーのテスト
  ├── package.json
  ├── tsconfig.json
  └── vitest.config.ts
```

## 開発環境のセットアップ

1. 依存関係をインストールします：

```bash
npm install
```

2. テストを実行します：

```bash
npm test
```

テストをウォッチモードで実行するには：

```bash
npm run test:watch
```

## 実装について

このプロジェクトでは、JSON 仕様に従ったパーサーを実装します。パーサーは以下の基本的な JSON 値をサポートします：

- null
- 真偽値（true / false）
- 数値
- 文字列（エスケープシーケンスを含む）
- 配列
- オブジェクト

## TDD での開発手順

1. テストファイル（`src/parser.test.ts`）には、JSON パーサーが満たすべき仕様を表すテストが記述されています
2. 各テストケースを一つずつ実装していきます
3. テストが成功するまで実装を改善します
4. すべてのテストが成功したら、実装は完了です

## JSON 仕様

このパーサーは[JSON 仕様](https://www.json.org/json-en.html)に準拠しています。実装する際は、以下の点に注意してください：

- JSON は、オブジェクト、配列、文字列、数値、真偽値、null の 6 種類の値をサポートしています
- JSON の文字列は二重引用符で囲む必要があります
- 数値は符号付き 10 進数で、指数表記もサポートしています
- オブジェクトのキーは文字列である必要があります
- 値の間のコンマの後にカンマを置くことはできません（末尾のカンマはサポートされていません）
- コメントはサポートされていません
